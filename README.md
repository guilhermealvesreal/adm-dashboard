# Dashboard Administrativo (SPA)

Dashboard administrativo desenvolvido com HTML, CSS e JavaScript puro, simulando uma aplicação real de gerenciamento com autenticação, CRUD e persistência de dados, tudo sem uso de frameworks.

Este projeto foi pensado para demonstrar domínio de lógica, organização de estado e arquitetura front-end.

## Funcionalidades
- Sistema de login
- Controle de sessão
- SPA (Single Page Application)
- Dashboard com métricas
- CRUD de usuários
- CRUD de tarefas
- Tema claro e escuro
- Persistência de dados no navegador
- Interface responsiva

## O que eu pratiquei neste projeto
- Manipulação avançada do DOM
- Controle de estado da aplicação
- Lógica de autenticação
- CRUD completo no front-end
- Persistência com localStorage
- Uso de variáveis CSS
- Organização de código
- Conceitos de SPA sem framework
